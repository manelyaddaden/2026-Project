# 2026-Project
My project for 2025–2026 is a mobile application that helps neighbors borrow and lend objects to each other. The goal of the app is to reduce waste, save money, and strengthen community connections. Instead of buying new items that people only use once or twice, users can borrow things like tools, school supplies, sports equipment, etc.

Lendify            Quick Start Guide
1. Open Terminal in VS Code In the top menu bar, click Terminal → New Terminal
2. Run These Commands

  npm install

Wait for installation to complete (1-2 minutes).
Then run:

  npx expo start

3. View the App
Press W to open the app in your web browser.
